NAME

dosfslabel − set or get MS−DOS filesystem label

SYNOPSIS

dosfslabel DEVICE [LABEL]

DESCRIPTION

dosfslabel set or gets a MS−DOS filesystem label from a given device.

If the label is omitted, then the label name of the specified device is written on the standard output. A label can’t be longer than 11 bytes.

OPTIONS

−h, −−help

Displays a help message.

−V, −−version

Shows version.
